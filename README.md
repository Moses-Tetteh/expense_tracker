*Project Title:*
Expense Tracker Web Application
Objective:
Build a basic web application where users can log their daily expenses, categorize
them, and view them in a structured list.
Core Requirements:
1. User Authentication: Implement Sign-Up, Login, and Logout using Django’s built-in
authentication system.
2. Expense Model: Create an Expense model with fields for user, amount, category,
date, and description.
3. CRUD Functionality:
● Add new expenses
● View list of expenses (user-specific)
● Edit existing expenses
● Delete expenses
4. Django Admin: Register the Expense model for backend management.
